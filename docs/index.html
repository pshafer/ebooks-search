Medical E-Books Documentation

About

I have developed this application as a side project to allow me to explore PHP Application development using the Symfony Framework and to sharpen my skills with Apache Solr, This application is developed in my free time and is mainly used a proof of concept and playground to help keep my skills sharp.

Purpose

The purpose of this application is to provide a simple search platform for E-Books (in this case Medical E-Books) for use by Academic Libraries. This application contains a publicly accessible front end faceted search utilizing Apache Solr and secure password protected administrative console to manage E-Books, Subject, and Vendors

Features

  • Public Faceted Search Interface that can perform Keyword search, Phrase Searching, and facet filter on Subject and Vendor
  • Secure application administrative console for manaing E-Books, Subject, and Vendors
  • Local Database Authentication
  • Search Index powered by Apache Solr
  • Application Developed in PHP using Symfony MVC Framework
  • MySQL Database
  • Database Interface utilizing PDO and Doctrine ORM
  • Twig Templates
  • CLI Interface for Bulk Data Imports and Bulk Data Indexing

Future Implementation Goals / Features

  • Search Result Pagination
  • CAS Authentication
  • REST API with Token based authentication
  • Adminisative Based User Management
  • User Favorite Lists with Sharing
  • LMS Integration
